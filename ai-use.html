AI Use Statement

AI tools were used as support during the project. The main use was to solve a problem I faced when the first website menu and HMI layout were too simple and did not match the course example. AI helped identify that the website should use real linked pages and that the HMI should look more like an industrial operator panel.

Problem Faced

The problem was that the first version of the website had menu buttons that did not work properly when opened as a static GitHub Pages site, and the HMI did not visually match the professor's HMI example. The correction was to create separate HTML pages for each menu item and add a desktop HMI with counters, alarms, pipeline boxes, temperature control, and a rejection log.

Most Relevant AI Prompt

Create a pencil production line project for Advanced Programming. Fix the website menu so every button opens a real page, and make the HMI look like an industrial panel with Start, Stop, Reset, machine state, alarms, temperature, pipeline stations, and rejection log.

Steps Taken to Correct AI Output

  • Changed the product to a pencil instead of a ball pen.
  • Checked that the line has at least four components.
  • Added defect reasons and rejection logging.
  • Created real website pages instead of only one-page anchors.
  • Kept README and run instructions in the repository and website.
  • Ran pytest to check the backend still works.
