Backend, Frontend, and HMI

The backend holds the production logic. The HMI shows the operator the live machine state and provides safe controls. The HMI must show useful information, make abnormal states clear, and never lie about machine state.

Pencil Backend

The backend uses Component, Pencil, Station, ComponentMaker, QualityControl, AssemblyStation, FinalInspection, Packaging, and ProductionLine classes. Defects are simulated and rejection reasons are stored.

MES / Telemetry

The producer script sends temperature, parts_total, and state to InfluxDB. Grafana displays these values in panels, similar to a small factory historian.
