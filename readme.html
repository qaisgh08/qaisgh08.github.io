What is inside the repository?

  • backend/pencil_line.py - object-oriented backend logic.
  • hmi.py and desktop_hmi/hmi.py - professor-style desktop HMI.
  • web_hmi/ - optional Flask web HMI.
  • telemetry/producer.py - sends data to InfluxDB.
  • docker-compose.yml - starts InfluxDB and Grafana.
  • website/ - this GitHub Pages website.
  • docs/ - PDF and DOCX documentation.
  • tests/ - pytest tests.

Why README is included

The assignment asks for a website and a PDF, but a README is still useful in the GitHub repository because it explains how to run the project quickly. This README page mirrors the repository README so the evaluator can understand the files from the website too.
