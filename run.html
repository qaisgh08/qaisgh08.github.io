Run Desktop HMI

pip install -r requirements.txt
python hmi.py

This opens the professor-style dark industrial HMI.

Run Flask Web HMI

python web_hmi/app.py

Open http://127.0.0.1:5000 in your browser.

Run InfluxDB + Grafana

docker compose up -d
python telemetry/producer.py

Open Grafana at http://localhost:3000. Login: admin / admin.
